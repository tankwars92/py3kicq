<h1 align="center">py3kicq</h1>
Библиотека для работы с протоколом OSCAR (использующемся в ICQ) на языке Python 3.

## Установка

1. Скачайте последнюю версию Python с [официального сайта](https://python.org/).
2. Скачайте весь репозиторий в отдельную папку.
3. Получите два UIN на [официальном сайте KICQ](http://kicq.ru/reg117623/) (или на другом сервере, где вы хотите воспользоваться данной библиотекой).
4. Используя первый UIN и пароль для него, измените файл с примером бота - [`echo_bot.py`](./echo_bot.py): найдите эти строки:
```python
_main_uin = 111111
_main_password = "password"

_test_uin = 111111
```
5. В переменную `_main_uin` запишите ваш первый UIN, в переменную `_main_password` - пароль от этого UIN. В переменную `_test_uin` запишите ваш второй UIN.
6. Теперь откройте файл [`echo_bot_test.py`](./echo_bot_test.py), найдите там эти строки:
```python
_test_uin = 111111
_test_password = "password"
```
7. В переменную `_test_uin` запишите ваш второй UIN, в `_test_password` - пароль от него.
8. Запустите скрипт `echo_bot_test.py`, используя команду
```
python3 echo_bot_test.py
```
9. Запустите скрипт  `echo_bot.py`, используя команду:
```
python3 echo_bot.py
```
10. Добавьте в свой контакт-лист контакт с первым UIN (полученным на шаге 3). Бот будет отвечать теми же самыми сообщениями, которые вы ему отправляете.
